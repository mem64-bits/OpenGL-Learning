# OpenGL-Learning

## Usage Guide 


#### 1. Clone repo with submodules
```
# git clone repo with submodules needed for external libraries (glad, glm, stb, etc.) 

git clone --recursive https://github.com/mem64-bits/OpenGL-Learning
```

#### 2. Install python packages jinja2 and glad to be able to generate glad context for configured OpenGL version in the root CmakeLists.txt.

### Windows 

 **if python is not installed get it from the official python website**
https://www.python.org/downloads/ 
(is confirmed to work on python 3.13 - 3.14)

### Linux

The preferred method to install python is through your distro's package manager.
(Many distros include a python installation by default)
